### Composer

To install the bindings via [Composer](http://getcomposer.org/), add the following to `composer.json`:

```

    {
      "type": "git",
      "url": "https://github.com/brandon14/amazon-vendor-direct-fulfillment-shipping-v20211228.git"
    }
  ],
  "require": {

  }
}
```
